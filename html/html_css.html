Vad är HTML?

HTML står för Hyper Text Markup Language. HTML är det vanligaste märkspråk för att skriva dokument som ska visas i webbläsaren. HTML kan kombineras med olika stilmaller såsom CSS och olika scriptspråk såsom Javascript för att skapa webbsidor.

Exempel på HTML

<!DOCTYPE= html>
<html>
 <body>
  <h1>Hello World!</h1>
 </body>
</html>

Vad är CSS?

CSS står för Cascading Style Sheets. CSS är en stilmall som används för att beskriva presentationen av dokument skriven i märkspråk såsom HTML.

Exempel på CSS

body {
background-color: white;
}
.picture {
height: 50px;
padding: 0;
}
