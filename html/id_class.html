Vad är ett ID?

Ett HTML ID är ett attribut som specificerar ett ID för ett HTML element. Det används oftast för att peka till en stilmall vanligtvis CSS. Genom ID och CSS kan man förändra ett eller flera elements position men ID används även för att sortera element. Grejen är att man inte kan ha flera ID på ett objekt vilket gör att man kan ha mer säkerhet på elementet där man vet att inget skriver över det. Man vill inte heller använda sig av ett ID på flera element.

Exempel på ID

 <body id="detta_är_ett_id">

Vad är en Class?

En HTML Class är väldigt lik ett HTML ID. Det är ett attribut som specificerar ett eller flera klassnamn för ett element. Det används oftast för att peka till en stilmall vanligtvis CSS. Genom en Class och CSS kan man förändra ett eller flera elements utseende, position men ID används även för att sortera element. Här kan man dock ha fler klasser på ett och samma element. En Class används gärna även på flera element samtidigt i skillnad från ID.

Exempel på en Class

 <body id="this_är_en_class">
