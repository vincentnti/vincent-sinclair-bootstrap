Vad är CDN?

CDN står för Content Delivery Network. Det används precis till det som det står för, det ger ut content via nätet. Med hjälp av CDN kan en användare som sitter i Sverige komma åt och fä bättre ladd tider av en sida som ursprungligen skapades på Hawaii. Det fungerar igenom användning av utspridda proxy servrar som håller och levererar statisk content via närmaste proxy server till användaren. Tidigare nämde vi att detta var ett alternativ till att importera bootstrap, vilket är snabbare vid produktion då användare var som helst får bättre prestanda.
