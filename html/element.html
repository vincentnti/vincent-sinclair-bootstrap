Vad är ett Element?

Ett HTML element är en typ av komponent av dokumentobjektsmodellen som används vid rendering av webbsidor som är strukturerade med HTML. Ett element brukar bestå av en starttagg och en sluttagg och kan ha olika attribut, namn och värden. Elementen kan användas för att strukturera, formatera och visa olika typer av innehåll. Element kan även hålla andra element.
