Vad är HTTP?

HTTP står för Hypertext Transfer Protocol. Det är en kommunikationsprotokoll, i stort sätt "regler" för ett kommunikationssätt. Det används för att överföra webbsidor på informationsnäverket WWW som står för World Wide Web. Ursprungliga användnings området av HTTP var att tillhandahålla en metod för att överföra HTML-sidor från webbservrar till webbklienter.

Vad är HTTPS?

HTTPS står for Hypertext Transfer Protocol Secure. Det är en kommunikationsprotokoll, i stortsätt "regler" för ett kommunikatinossätt. HTTPS är ett nyare sätt att skicka information, det krypterar informationen som sänds mellan dig och webbservern för att skapa en säker uppkoppling. Man kan föreställa sig kryptering som ett rör mellan dig och webbservern vilket gör att utestående inte kan se vad som pågår i röret.
